Coronium SkyTable

A performant and secure user scoped table datastore and client for use with Corona.

Available for installation as a DigitalOcean droplet or Amamzon EC2 instance.

Plugin Required

You will need the Coronium SkyTable Plugin available in the Corona Marketplace, in addition to a running server install.
